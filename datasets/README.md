# Datasets

Dataset should be downloaded separately. Place your dataset with the following structure:

```
datasets/
├── train/
│   ├── images/
│   └── masks/
├── val/
│   ├── images/
│   └── masks/
└── test/
    ├── images/
    └── masks/
```

This project uses the **Bijie Landslide Dataset**. Download from:
https://www.kaggle.com/datasets/your_dataset
